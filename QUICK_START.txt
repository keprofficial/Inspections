STEP 1: Install Flutter
------------------------
Visit: https://flutter.dev/docs/get-started/install
Download and extract Flutter SDK

STEP 2: Setup Environment
--------------------------
Add Flutter to your PATH:
- Windows: C:\flutter\bin
- Mac/Linux: ~/flutter/bin

STEP 3: Verify Installation
----------------------------
Open Terminal/CMD and run:
  flutter --version

STEP 4: Get Dependencies
------------------------
cd c:\Users\purus\OneDrive\Documents\A\kepr
flutter pub get

STEP 5: Connect Device or Emulator
-----------------------------------
For Android:
  - Open Android Emulator (Android Studio)
  OR
  - Connect physical Android device via USB

For iOS:
  - Open iOS Simulator (Mac only)
  OR
  - Connect physical iPhone via USB

STEP 6: Run the App
-------------------
flutter run

STEP 7: Navigate the App
------------------------
Screen Flow:
  1. Sign Up (pre-filled) → Check terms → Generate OTP
  2. Property Details → Enter details → Continue
  3. Dashboard → Click inspection area
  4. Area Details → Click checklist item
  5. Item Details → Set severity → Mark Complete
  6. Profile → View stats, settings, logout

TROUBLESHOOTING
===============

If "flutter: command not found":
  → Add Flutter SDK to PATH
  → Restart terminal/IDE

If "No connected devices":
  → Open Android Emulator or iOS Simulator
  → Or connect physical device via USB

If "Build failed":
  → Run: flutter clean
  → Run: flutter pub get
  → Run: flutter run

If "Gradle issues":
  → Run: flutter clean
  → Run: flutter run (with internet connection)
